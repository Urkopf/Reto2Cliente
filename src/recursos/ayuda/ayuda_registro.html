Manual de Usuario - Registro

Introducción

Este manual explica cómo registrar un nuevo usuario en la aplicación de forma sencilla. Siga las instrucciones detalladas para completar el proceso de registro.

Rellenar Información Personal

Complete los siguientes campos:

  • Nombre: Escriba su nombre completo.
  • Apellidos: Proporcione sus dos apellidos.
  • Email: Ingrese un correo electrónico válido.
  • Contraseña: Cree una contraseña segura.
  • Confirmar contraseña: Repita la contraseña para confirmarla.

Seleccionar Tipo de Usuario

Elija si desea registrarse como cliente o trabajador marcando la opción correspondiente:

  • Cliente: Complete los campos adicionales como sector y teléfono.
  • Trabajador: Seleccione un departamento y categoría.

Opciones Avanzadas

Configure las siguientes opciones según sea necesario:

  • Activo: Marque esta opción si el usuario estará habilitado para iniciar sesión.
  • Sector y Teléfono: Obligatorios para clientes.
  • Departamento y Categoría: Requeridos para trabajadores.

Finalizar Registro

Una vez completados todos los campos:

  • Haga clic en el botón "Registrar" para enviar el formulario.
  • Si desea cancelar, haga clic en el botón "Cancelar".

Con estas instrucciones, podrá registrar un usuario de manera eficaz. Si tiene dudas, contacte al soporte técnico para obtener ayuda adicional.
